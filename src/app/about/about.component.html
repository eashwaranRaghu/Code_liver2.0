CodeLiver

CodeLiver is a web application that provides various tools and infrastructure aimed to support coders in real-time collaboration.

Editor: stays synced across all nodes and any changes anywhere is reflected in real-time.

Rooms: users may enter any room and start using editor without affecting other rooms.

Screen Sharing: only one person can share screen or camera at a time and others can watch the stream.

Drop Zone: can be used to share any files less then 15MB size, which will be available to everyone.
